# Music-recommendation

Целью проекта состоит в создании системы музыкальных рекомедаций для пользователей стриминговых сервисов. Проект востребован, потому что рынок музыкальных сервисов растет с каждым годом.

# Данные
Для решения задачи собираюсь использовать Million Song Dataset(http://millionsongdataset.com), как можно понять из названия, он содержит данные о миллионе треков, для сравнения, в Apple Music сейчас примерно 100 миллионов треков, в Spotify около 50, в Яндекс.Музыке 76, то есть датасет адекватен по размеру в сравнении с этими сервисами.
